(A) Preorder 顯示解析
錯！
Pre = Previous (前面)。
Preorder 是「根 → 左 → 右」。根在最前面先被拜訪。
(B) Inorder 顯示解析
正確！
In = Inside/Intermediate (中間)。
Inorder 正是「左 → 根 → 右」。根夾在左右子樹的中間。
(C) Postorder 顯示解析
錯！
Post = Posterior (後面)。
Postorder 是「左 → 右 → 根」。根在最後面才出現。
(D) Level-order 顯示解析
錯！
Level-order 是「層序走訪」。
規則是：第1層看完看第2層，由左而右。不看根在中間還是旁邊。

老師的私房筆記

★ 記憶口訣：看「根」在哪裡

  • Preorder：根在前 (根→左→右)
  • Inorder：根在中 (左→根→右)
  • Postorder：根在後 (左→右→根)

⚠️ 常見陷阱

  • 只背英文名稱，卻忘了 Root 的位置。
  • 搞混 Inorder (中) 和 Preorder (前) 的順序。
  • 看到「由上到下」就選 Level-order，但題目如果有規定「左右順序」就要小心。

準備好揭曉答案了嗎？

B

正確答案：Inorder

理由：題目標示「左子樹 → 根 → 右子樹」，
根夾在中間，這正是 Inorder (中序) 的定義！
