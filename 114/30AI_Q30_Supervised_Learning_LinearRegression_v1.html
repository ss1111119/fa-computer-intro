什麼是監督式學習？
一題就會判斷

Machine Learning 基礎必考觀念

題號：30
題目
30. 下列哪一項屬於監督式學習（Supervised Learning）？
(A) K-means
(B) PCA
(C) Linear Regression
(D) Apriori
C. 關鍵字教學（點擊按鈕）

點選下方關鍵字，題目對應處會亮起，並顯示白話解釋。

請點擊上方按鈕查看說明。

翻譯：「哪一個演算法是『老師先給答案，電腦照答案學』的那一種？」

重點：你能不能分清楚「有沒有標準答案（標籤）」，來判斷是不是監督式學習。看到「預測數值」或「分類別」通常是監督式；看到「分群」或「找關聯」通常是非監督式。

E. 兩大學習類型對照

點擊演算法名稱，看看它屬於哪一邊：

監督式學習
(有正確答案 / Label)
  • Linear Regression
  • Logistic Regression
  • Decision Tree
非監督式學習
(無正確答案 / Unlabeled)
  • K-means
  • PCA
  • Apriori
請點擊上方演算法名稱
F. 選項逐一解析
(A) K-means

錯誤。
K-means 是「分群」演算法，資料沒有標籤，屬於非監督式學習。

(B) PCA

錯誤。
PCA (主成分分析) 是用來「降維」的，不需要答案標籤，屬於非監督式學習。

(C) Linear Regression

正確！
Linear Regression (線性迴歸) 需要利用「已知答案」(例如過去的坪數與房價) 來訓練模型，預測未來的數值，屬於監督式學習。

(D) Apriori

錯誤。
Apriori 是用來找「關聯規則」(例如買麵包通常會買牛奶)，資料本身沒有預設的標準答案，屬於非監督式學習。

G. 隨堂小測驗

1. 資料「沒有」標記正確答案，是哪一類學習？

2. 想預測明天的「股價」(連續數值)，最可能用哪種？

3. 想要把全校學生依興趣自動分成 5 群，該用？

H. 老師的獨家筆記
【記憶口訣】
  • ★ 有答案找監督，沒答案找非監督
  • ★ 預測、分類 → 監督 (Supervised)
  • ★ 分群、關聯 → 非監督 (Unsupervised)
【常見陷阱】
  1. 看到英文縮寫就亂選 (例如 PCA 看起來很厲害就選它，錯！)。
  2. 把「關聯規則 (Association)」誤以為是監督式，其實它是非監督。
  3. 分不清 Regression (預測數值) 與 Classification (預測類別)，但這兩個其實都是監督式學習。

正確答案：(C)

Linear Regression (線性迴歸)

一句話理由：
Linear Regression 使用「有標記答案」的歷史資料來訓練模型，這正是監督式學習的定義。
